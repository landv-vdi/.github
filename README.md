
虚拟桌面基础设施（VDI）是一种用于集中创建和管理虚拟桌面的软件工具。 虚拟桌面是模拟真实台式计算机的软件应用程序。 VDI 技术在中央服务器上创建虚拟桌面，远程用户可以通过互联网从任何实体机访问此桌面。 VDI 可快速高效地设置许多虚拟桌面，以提供对内部业务应用程序和服务的安全远程访问。
